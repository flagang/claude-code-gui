# Claude Code GUI

一个 macOS 原生风格的 Claude Code 桌面客户端，为 Claude Code CLI 提供图形化管理界面。

## 功能

### 多会话管理
- 基于 node-pty 启动独立的 Claude Code 进程，每个会话拥有独立 PTY
- 顶部标签栏快速切换会话
- 左侧侧边栏展示会话详情（工作目录、模型、启动时间）
- 支持新建会话（默认使用当前项目目录）和恢复已有会话（`--resume`）
- 自动发现 `~/.claude/sessions/` 下的历史会话

### 终端交互
- 基于 xterm.js 的全功能终端渲染，支持 256 色和 ANSI 转义序列
- 自动适配窗口大小（addon-fit）
- 可点击链接（addon-web-links）
- 会话切换时保留终端回滚历史

### 模型切换
- 工具栏模型选择器，支持 opus / sonnet / haiku
- 新建会话时通过 `--model` 参数指定模型
- 运行中会话通过 `/model <name>` 命令切换

### 命令面板
- 快捷键 `Cmd+Shift+P` 呼出
- 快速执行常见斜杠命令：`/help`、`/clear`、`/compact`、`/model`、`/config` 等

### 配置面板
- 可视化编辑 `~/.claude/settings.json` 和 `settings.local.json`
- 环境变量键值对编辑器
- 权限规则（allow 列表）管理
- 配置修改后直接写入文件

### macOS 原生体验
- 隐藏标题栏 + 毛玻璃侧边栏（vibrancy）
- 原生菜单栏 + 键盘快捷键
- 深色/浅色主题自动跟随系统
- SF Mono 等宽字体终端渲染

## 项目结构

```
src/
  main/                  # Electron 主进程
    index.ts             # 窗口生命周期 + IPC 注册
    pty.ts               # node-pty 会话管理（spawn/write/resize/kill）
    sessions.ts          # 历史会话发现与存活检测
    config.ts            # 读写 ~/.claude/settings 配置
    menu.ts              # macOS 原生菜单 + 快捷键
  preload/
    index.ts             # contextBridge 暴露 IPC API
  renderer/              # React 渲染进程
    App.tsx              # 主布局（侧边栏 + 标签栏 + 终端 + 工具栏）
    store.ts             # Zustand 全局状态
    components/
      Sidebar.tsx        # 会话列表侧边栏
      TabBar.tsx         # 顶部标签栏
      TerminalPane.tsx   # xterm.js 终端容器
      Toolbar.tsx        # 底部工具栏
      ModelSelector.tsx  # 模型下拉选择
      CommandPalette.tsx # 命令面板
      ConfigPanel.tsx    # 配置面板
      EnvVarEditor.tsx   # 环境变量编辑
      PermissionList.tsx # 权限规则管理
  shared/
    types.ts             # 跨进程共享类型 + IPC 通道常量
```
